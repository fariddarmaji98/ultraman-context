# Project Overview

## Tujuan Dokumen
Sumber kebenaran utama untuk AI assistant dan developer.
Baca file ini + file terkait sebelum membuat perubahan apapun.

## Konsep Utama
- **Atomic Design** strict: atoms → molecules → organisms → templates
- **Reusable Component** — cek `COMPONENTS.md` sebelum buat component baru
- **AI-native workflow** — semua keputusan teknis terdokumentasi
